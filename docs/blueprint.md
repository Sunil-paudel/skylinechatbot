# **App Name**: AetherChat

## Core Features:

- Send Message: Allows users to send text messages to the AI assistant.
- Receive AI Response: Receives and displays responses generated by the AI assistant.
- View Chat History: Displays a scrollable history of the current conversation.
- New Chat: Starts a new, empty conversation, clearing the existing chat history.
- Intelligent Context Handling: The AI tool remembers the context of the ongoing conversation to provide more relevant and coherent responses.

## Style Guidelines:

- Primary color: Light blue (#90CAF9) to create a calm and modern feel.
- Background color: Off-white (#F5F5F5), slightly desaturated from the primary, for a clean interface.
- Accent color: A darker blue (#64B5F6) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font, for clear readability.
- Module-based design for a structured and scalable layout.
- Use simple, outline-style icons for actions and navigation.
